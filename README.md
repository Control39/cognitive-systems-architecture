# 🧠 Portfolio System Architect

> **Документальный фильм о рождении новой профессии: Архитектор когнитивных систем**

---

## 🎯 Цель

Представить **единый, целостный, живой репозиторий `portfolio-system-architect`**, который:
- Станет **доказательством концепции новой роли в IT**
- Покажет, как **системное мышление преображается в архитектурную компетенцию**
- Будет основой для **подачи на грант Sourcecraft Open Source**
- Ответит на главный вопрос:
  > «Кто ты?» →
  > «Я — архитектор когнитивных систем. Я проектирую системы, в которых человек управляет ИИ для создания новых форм экспертизы.»

---

## 🧩 Архитектура системы

Этот репозиторий — не просто код, а **структурированное повествование о моем пути** от нуля к новой профессии. Он состоит из пяти основных глав:

```
cognitive-architect-manifesto/
├── 01_JOURNEY/                    → **Путь** (доказательство эволюции)
│   ├── 01_excel-to-methodology/   → Как Excel-таблица стала IT-Compass
│   ├── 02_chaos-to-system/       → Как хаос диалогов стал RAG-системой
│   └── 03_projects-to-ecosystem/ → Как 7 проектов сложились в единую экосистему
│
├── 02_METHODOLOGY/                → **Ядро** (то, что создано)
│   ├── it-compass/               → Методология объективных маркеров (ядро)
│   ├── arch-compass/             → Архитектурный фреймворк
│   ├── markers/                  → Маркеры компетенций по областям
│   └── career-development/        → Система развития карьеры
│
├── 03_EVIDENCE/                  → **Доказательства** (что работает)
│   ├── rag-system/                → Автоматизированный анализ тысяч файлов
│   ├── reasoning-loop/          → Циклы ИИ-анализа для извлечения инсайтов
│   └── portfolio-generator/      → Генерация портфолио из собранных доказательств
│
├── 04_ARTIFACTS/                 → **Артефакты** (результаты)
│   ├── case-studies/             → Конкретные кейсы применения (из диалогов)
│   ├── demos/                    → Интерактивные демонстрации
│   └── grants/                   → Подготовленные материалы для гранта
│
└── 05_MANIFEST/                 → **Манифест** (кто я теперь)
    ├── README.md                → Визитка архитектора: «Я создала новую профессию»
    ├── ARCHITECTURE.md         → Архитектурная схема всей экосистемы
    └── METHODOLOGY.md          → Полное описание авторской методологии
```

## 🚀 Начало работы

1. **Клонирование репозитория**
   ```bash
   git clone https://github.com/leadarchitect-ai/portfolio-system-architect.git
   cd portfolio-system-architect
   ```

2. **Установка зависимостей**
   ```bash
   pip install -r requirements.txt
   ```

3. **Генерация карты знаний**
   ```bash
   python scripts/generate_obsidian_map.py
   ```

4. **Генерация веб-сайта**
   ```bash
   python scripts/generate_website.py
   ```

5. **Ежедневная автоматизация**
   ```powershell
   .\scripts\run_daily.ps1
   ```
