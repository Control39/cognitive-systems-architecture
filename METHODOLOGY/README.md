# METHODOLOGY

Папка содержит методологию когнитивного архитектора, объективные маркеры компетенций и философские основы.

## Содержание

- `competency-markers.md` — объективные маркеры компетенций для оценки архитектурных решений.
- `cognitive-architect-philosophy.md` — философия когнитивного архитектора, принципы системного мышления.
- `it-compass-methodology.md` — методология IT‑Compass (связь с реализацией в `apps/it-compass/`).

## Назначение

Эта папка отражает методологическую базу портфолио, демонстрируя не только технические навыки, но и глубину архитектурного мышления, что критично для грантов и собеседований с HR.

## Связи

- Реализация методологии IT‑Compass находится в `apps/it-compass/`.
- Принципы используются в компонентах Cloud‑Reason, Arch‑Compass‑Framework.
- Доказательная база — в `EVIDENCE/`.
