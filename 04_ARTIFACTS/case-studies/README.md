# Кейсы и исследования

Здесь собраны аналитические кейсы и исследования, демонстрирующие применение методологии архитектора когнитивных систем.

## Список кейсов

- **ADR_it-compass_architecture.md** — архитектурное решение по интеграции IT-Compass.

## Назначение

Кейсы служат доказательной базой для методологии и показывают, как системное мышление применяется на практике.

## Как добавить новый кейс

1. Создайте Markdown-файл в этой папке.
2. Опишите контекст, проблему, решение и результаты.
3. Укажите связи с другими компонентами системы.
