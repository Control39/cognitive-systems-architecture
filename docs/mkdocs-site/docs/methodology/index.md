# Methodology

## Системное мышление архитектора

### Основные методологии
- **Soft Systems Methodology (SSM)**: Анализ сложных проблем через rich pictures
- **Viable System Model (VSM)**: Модели жизнеспособных систем
- **System Dynamics**: Динамика систем и петли обратной связи

### Ключевые принципы проекта
1. **Человек управляет ИИ** - 60% предложений ИИ отклонено
2. **Самоанализ** - ИИ анализирует ИИ через RAG loops
3. **Метрики доверия** - Логи human-decisions

**Подробнее**: См. [ADR-001](adr/ADR-001-system-thinking-methodology.md) и исходники в `02_METHODOLOGY/`.
